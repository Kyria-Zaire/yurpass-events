# Tech Stack - YURPASS MVP

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + Shadcn/UI (Modern, clean UI)
- **Language:** TypeScript
- **Form Handling:** React Hook Form + Zod
- **Database/Backend:** Supabase (Fastest for MVP) or simple Airtable API.
- **Emails:** Resend (Clean API for transactional emails)
- **Deployment:** Vercel
