# YURPASS Design System

## Colors
- **Background:** #FFFFFF (Pure White) - No gradients.
- **Foreground/Text:** #000000 (Pure Black).
- **CTA/Buttons:** Background #000000, Text #FFFFFF.
- **Cards/Sections:** Border 1px solid #000000 or light grey, Background #FFFFFF.

## Aesthetics
- **Style:** Minimalist, High-contrast, Bold.
- **Typography:** Sans-serif, bold headings.
- **Images:** Large, immersive photography (Event lifestyle). 
- **Radius:** Small (4px or 8px) for a "tech" feel, not too rounded.
