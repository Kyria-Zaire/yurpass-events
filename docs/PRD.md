# PRD - YURPASS MVP (Event Launch)

## Project Overview
**Product Name:** YURPASS (Event wing of Yunicity)
**Objective:** Validate demand for local events through a "Concierge MVP".
**Goal:** Manage the first event on Jan 17th (Raclette Party, 15 seats).

## Target Audience
- Young locals in Reims (Yunicity users).
- People looking for curated, small-scale social interactions.

## Core Features (V1 - Jan 17)
1. **Landing Page:** - Pitch of the YURPASS concept.
   - Event details (Date, Place: Reims, Theme: Raclette).
   - "Apply for a seat" CTA.
2. **Application Form:**
   - Name, Social Handle (Instagram/Yunicity), Motivation.
3. **Admin/Back-office (Manual):**
   - Collect applications.
   - Status: "Pending", "Accepted", "Rejected".
4. **Notification System:**
   - Email confirmation 48h after application.

## User Flow
1. User lands on page -> Reads about YURPASS.
2. User applies -> Sees "Success, we'll get back to you in 48h".
3. Admin selects -> User receives "You're in!" email with location.
